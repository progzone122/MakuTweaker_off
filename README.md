# MakuTweaker [RU]
Самый дружелюбный и полезный твикер для Windows.
В этой программе вы можете гибко настроить и персонализировать систему, а так же управлять процессами и посмотреть подробную информацию о характеристиках вашего ПК.
Также, вы можете выполнить быструю настройку Windows, управлять установкой системных компонентов и избавиться от системного мусора.
В MakuTweaker доступно несколько тем оформления, благодаря которым вы можете настроить программу на свой вкус. Дизайн построен на базе MicaWPF для полной интеграции с дизайном Windows 11.

# Технические подробности
Проект разработан в Visual Studio 2022.
Работает на платформе .NET Core 8.0.
Открытый исходный код для сообщества
Для корректной работы в IDE, требуются права администратора.
Чтобы визуальный конструктор (Designer) корректно отображал элементы и подтянул кэш, проект необходимо собрать перед первым открытием форм в редакторе.

# Ограничения версии
В данной версии исходного кода отсутствует страница "Активатор Windows" по юридическим соображениям.


# MakuTweaker [EN]
The most user-friendly and useful tweaker for Windows.
With this program, you can flexibly configure and personalize the system, manage processes, and view detailed information about your PC’s specifications.
You can also perform a quick Windows setup, manage the installation of system components, and remove system bloatware.
MakuTweaker offers several interface themes that allow you to customize the program to your taste. The design built on the MicaWPF framework for full integration with the Windows 11 design language.

# Technical Details
IDE: Developed in Visual Studio 2022.
Framework: Powered by .NET 8.0.
Publicly available source code for the community.
Administrator rights are required for correct operation within the IDE.
To ensure the Visual Studio Designer displays elements correctly and populates the cache, you must build the project before opening forms in the editor for the first time.
